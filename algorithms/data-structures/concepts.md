## Concepts

### Primitive types

Boolean `true, false`

Character `a, b, f, k ,g, ...`

Integer `-33, -1, 0, 1, 12, 125, ...`

Float `-0.12, 0.43, 4.3544, 30349.3423, ...`

### Complex types

Array is a collection of elements `[1, 3, 5, 7]`

Matrix is a 2-dimensional array of elements `[[1, 3], [5, 7]]`

String is a collection of characters `"My namE is X"`
